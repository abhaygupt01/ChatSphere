# ChatSphere 💬

Real-time chat application built using:

- React.js
- Node.js
- Express.js
- MongoDB
- Socket.io
- Tailwind CSS

## Features 🚀

- User Authentication
- Real-time Messaging
- Online/Offline Status
- Responsive UI
- Socket.io Integration

## Run Locally

### Client

```bash
cd client
npm install
npm run dev

## server
cd client/server
npm install
npm run dev
